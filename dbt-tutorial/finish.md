# Поздравляем!

Вы завершили курс по dbt. Вот что вы изучили:

- **dbt init** — создание нового проекта
- **dbt debug** — проверка подключения к базе данных
- **dbt run** — сборка моделей (представлений/таблиц) из SQL-файлов
- **dbt compile** — компиляция Jinja-шаблонов в SQL
- **dbt docs generate** — генерация документации
- **{{ ref() }}** — управление зависимостями между моделями

## В следующем уроке

- Создание таблиц вместо представлений (`materialized='table'`)
- Написание тестов качества данных
- Макросы и переиспользуемые тесты
- Настройка sources для ссылок на исходные данные
- Инкрементальная загрузка данных
- Создание snapshots
