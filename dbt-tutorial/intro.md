# Введение в dbt

В этом курсе вы:

1. Запустите dbt и познакомитесь с его CLI
2. Проверите версию dbt
3. Инициализируете новый проект dbt с подключением к DuckDB
4. Создадите SQL-модель и запустите её командой `dbt run`
5. Добавите тесты качества данных и запустите их командой `dbt test`

В фоновом режиме выполняется скрипт установки dbt. К началу первого упражнения всё будет готово.
