# Введение в dbt

В этом курсе вы:

1. Познакомитесь с CLI dbt
2. Инициализируете проект dbt с подключением к DuckDB
3. Создадите SQL-модели и запустите их командой `dbt run`
4. Научитесь использовать Jinja-шаблоны и функцию `ref()`
5. Сгенерируете документацию командой `dbt docs`

В фоновом режиме выполняется скрипт установки dbt. К началу первого упражнения всё будет готово.

**Важно:** начиная с шага 4, упражнения являются последовательными и не могут быть пропущены. Каждый шаг зависит от результатов предыдущего.
