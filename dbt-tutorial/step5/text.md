# Настройка профиля проекта

Вы инициализировали проект dbt. Теперь необходимо настроить профиль подключения — файл `profiles.yml`, который определяет, где находится база данных. dbt использует YAML-файлы для конфигурации — важно соблюдать отступы и форматирование.

## Задание

Откройте файл `~/.dbt/profiles.yml`, заполните пропуски и проверьте подключение к базе данных.

<details>
<summary>Решение</summary>

Замените <code>_____</code> в первой строке на <code>nyc_yellow_taxi</code>, в строке <code>type:</code> на <code>duckdb</code>. Затем <code>cd ~/nyc_yellow_taxi && dbt debug</code>.
