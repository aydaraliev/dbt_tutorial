# Введение в dbt

В этом курсе вы:

1. Запустите dbt и познакомитесь с его CLI
2. Инициализируете новый проект dbt с подключением к PostgreSQL
3. Создадите SQL-модель и запустите её командой `dbt run`
4. Добавите тесты качества данных и запустите их командой `dbt test`

В фоновом режиме выполняется скрипт установки PostgreSQL и dbt. К началу первого упражнения всё будет готово.
